# Claude Code 使用规范

## 语言规范

**重要：必须使用中文回复所有消息。**

- 所有回复、说明、注释都必须使用中文
- 代码中的变量名、函数名等保持英文
- 错误信息和日志可以保持原文
- 文档和说明必须使用中文

## 服务启动规则

**重要：除非用户明确要求，否则不要自动启动任何服务。**

### 禁止自动启动的操作

- 不要自动运行 `npm run dev`
- 不要自动运行 `npm start`
- 不要自动启动数据库服务
- 不要自动启动任何后台进程

### 何时可以启动服务

只有在以下情况下才能启动服务：

1. 用户明确要求启动服务（例如："启动开发服务器"、"运行项目"）
2. 用户要求测试功能，且需要运行服务才能测试
3. 用户明确表示需要查看运行效果

### 正确的工作流程

1. **代码修改** - 直接进行代码修改，无需启动服务
2. **询问用户** - 完成修改后，询问用户是否需要启动服务进行测试
3. **等待确认** - 等待用户明确指示后再启动服务

## 示例

### ❌ 错误做法

```
我已经完成了代码修改，现在启动开发服务器来测试...
[自动运行 npm run dev]
```

### ✅ 正确做法

```
我已经完成了代码修改。如果您需要测试，我可以帮您启动开发服务器。
```

## 其他注意事项

- 完成任务后，不要自动关闭或重启服务
- 如果需要安装依赖，先询问用户
- 保持对用户工作环境的尊重，避免不必要的干扰
